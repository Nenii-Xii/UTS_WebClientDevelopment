# UTS Web Client Development

Repository ini dibuat untuk memenuhi tugas Ujian Tengah Semester (UTS) mata kuliah **Web and Cloud Development (WCD)**. Di dalamnya terdapat implementasi fungsi-fungsi sederhana menggunakan **JavaScript** dan pengelolaan proyek menggunakan **Git & GitHub**.

## Daftar Fungsi

Semua fungsi disimpan dalam folder `functions/`:

1. **fahrenheitToCelsius.js**
   - Mengubah suhu dari Fahrenheit ke Celsius.
2. **kmToMeter.js**
   - Mengonversi jarak dari kilometer ke meter.
3. **isOdd.js**
   - Mengecek apakah sebuah bilangan ganjil.
4. **removeChar.js**
   - Menghapus semua karakter tertentu dari sebuah string.
5. **isPalindrome.js**
   - Mengecek apakah string merupakan palindrome (sama dibaca dari depan maupun belakang).
